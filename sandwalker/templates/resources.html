The Sandwalker Database

The database of the the Sand Walker project contains rewards entries from all pocket accounts since block 0, it is structured as follows:

  • id, a unique internal identifier for the reward,
  • account, a pocket account,
  • block, the block height,
  • reward, the reward in µPkt.

The database can be either rebuilt by replaying the blockchain using tools from the public repository, or by downloading it below.
